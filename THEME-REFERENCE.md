# ReeveOS Portal Theme Reference
## Pulled directly from portal/src/config/theme.js
## EVERY component in the AI Builder MUST use these values

### Brand
- Primary: `#111111` (Rich Black)
- Gold: `#C9A84C`
- Gold Light: `#D4B76A`
- Gold Faint: `#F8F0DC` (gold tinted backgrounds)
- White: `#FFFFFF`

### Text
- Primary: `#111111` (headings, body)
- Secondary: `#374151` (subheadings)
- Muted: `#565B66` (labels, less important text)
- Light: `#7D828C` (placeholders, subtle text)
- Inverse: `#FFFFFF` (text on dark backgrounds)

### Backgrounds
- Page: `#FFFFFF` (main background)
- Card: `#FFFFFF` (cards, panels)
- Subtle: `#F9FAFB` (slightly off-white sections)
- Muted: `#F3F4F6` (hover states, zebra rows)
- Dark: `#111111` (sidebar rail, terminal)

### Borders
- Default: `#C4C8CF` (standard card borders)
- Light: `#E5E7EB` (subtle dividers)
- Input: `#D1D5DB` (form input borders)
- Focus: `#111111` (focused inputs)

### Status
- Success: `#22C55E`
- Error: `#EF4444`
- Warning: `#F59E0B`
- Info: `#3B82F6`

### Radius
- sm: 6px
- md: 8px
- lg: 12px

### Font
- Family: 'Figtree', system-ui, sans-serif
- Code: 'JetBrains Mono', monospace

### THE PORTAL IS LIGHT THEME
White backgrounds, gray borders, black text. 
The ONLY dark element is the sidebar rail and the terminal panel.
Everything else is white/light gray.
